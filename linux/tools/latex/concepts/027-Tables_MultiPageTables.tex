% Description: Multi-page Table

% Note
% 1. By default, the table will be cropped at the bottom of the page.
% 2. The package longtable provides a convenient way to make tables span multiple pages.
% 3. The longtable environment replaces both table and tabular environments or rather combines both of them into a
%    single environment.
% 4. Use \begin{longtable}[POSITION_ON_PAGE]{ALIGNMENT} which will automatically split between pages, if it has too many
%    rows.
% 5. The header will be repeated on every page, but without the caption. This is because the commands \endfirsthead and
%    \endhead to my table, where everything written before \endfirsthead declares the header for the first page the
%    table occurs on and everything between \endfirsthead and \endhead denotes the header, which should be repeated on
%    every following page. If the header is not required on the following pages, one can simply remove everything in
%    between \endfirsthead and \endhead including the \endhead command.

\documentclass{article}

\usepackage{siunitx}
\usepackage{booktabs}
\usepackage{longtable}                                          % To display tables on several pages.

\sisetup{
  round-mode          = places,     % Rounds numbers
  round-precision     = 2,          % to 2 places
}
