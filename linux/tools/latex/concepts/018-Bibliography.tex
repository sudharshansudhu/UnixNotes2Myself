% 3.  For BibTeX put the \bibliography statement in your document, for BibLaTeX in the preamble.

% 5.  BibLaTeX chooses the style as an option like: \usepackage[backend=bibtex, style=verbose-trad2]{biblatex}.

% 7.  The \autocite command takes the page number as an option: \autocite[NUM]{}.
% 8.  Combile .bib file using Texmaker by clicking Tools - BibTex.
% 9.  Compile .bib file using command line
%       pdflatex 015-Bibliography.tex   # Tell bibtex what literature we cited in our paper.
%       bibtex 015-Bibliography         # Without extension .bib. This will .aux file generated by above command.
%       pdflatex 015-Bibliography.tex
%       pdflatex 015-Bibliography.tex   # Compile twice.
